# MSP430_scoreboard
# 篮球比赛计分器

## 项目简介

本项目基于 **MSP430F6638** 单片机设计并实现了一个篮球比赛计分器系统，具备比分显示、进攻倒计时、全场倒计时、得分记录、队员得分统计等功能，支持通过按键进行交互控制，适用于篮球比赛的实时计分与计时管理。

## 功能描述

- 使用 **LCD** 显示两支队伍的当前比分
- 支持为队伍分别添加 **1、2、3 分**（通过矩阵按键实现）
- 显示当前进攻队伍的进攻时间倒计时
- 支持切换进攻方向、切换 LCD 显示内容（比分 / 进攻倒计时 / 队员得分）
- 倒计时归零时，**红色 LED 灯闪烁**
- 显示当前半场情况及全场倒计时（格式：xx.xx）
- 自动记录比赛过程中的得分时间表
- 统计每位队员的得分情况，并支持通过 LCD 查询

## 硬件平台

- 单片机：MSP430F6638
- 显示模块：LCD 显示屏、LED 数码管
- 输入模块：矩阵键盘、独立按键
- 指示灯：红色 LED（P4.4）

## 软件工具

- 开发环境：IAR Embedded Workbench
- 语言：C 语言

## 按键功能说明

按键功能：           
主按键1：切换 LCD 显示信息
主按键2：切换进攻方向、切换队员得分情况表
A：对 A 队加分
B：对 B 队加分
C：上切得分时间表
D：下切得分时间表
E：切换 LED 显示信息
F：暂停计时
1-3：选择得分数、选择得分队员序号
4-0：选择得分队员序号

## 系统结构

- 使用 **Timer_A** 模块实现 1 秒定时中断，驱动进攻倒计时与全场倒计时
- 通过结构体 `Goalrecord` 和 `Matescore` 分别记录队伍得分时间表和队员得分情况
- 使用 `pageLCD` 和 `pageLED` 变量控制 LCD / LED 的显示内容切换
- 主循环中不断检测按键状态，执行相应操作并刷新显示

## 运行效果

- 开机显示进攻倒计时与全场倒计时
- 按下主按键 1 可依次切换显示：进攻倒计时 → 比分 → 队员得分
- 按下 E 键切换 LED 显示：全场倒计时 ↔ 队伍得分时间表
- 倒计时结束红色 LED 闪烁
- 支持多步加分操作（先选队员，再选分数）
